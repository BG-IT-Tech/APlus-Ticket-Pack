# CompTIA A+ Beginner GitHub Support Ticket Pack

This repository is a starter pack for practical first-line support exercises using GitHub Issues and markdown files.

## What is included
- 20 beginner-friendly IT support ticket scenarios
- student response template
- instructor answer key pack
- suggested GitHub issue templates
- evidence files that can be attached to tickets
- labels and workflow suggestions

## Suggested classroom workflow
1. Instructor opens a ticket as a GitHub Issue using the scenario title and body.
2. Students review the ticket and attached evidence in the repository.
3. Students complete `templates/student-response-template.md`.
4. Students submit answers through:
   - a pull request
   - an issue comment
   - a new branch with their completed response
5. Instructor marks the response against the answer key.

## Mapping to CompTIA A+
These scenarios support:
- basic troubleshooting methodology
- hardware and peripherals
- Windows and mobile devices
- networking fundamentals
- operational procedures
- basic security awareness

## Folder overview
- `.github/ISSUE_TEMPLATE/` - issue templates for reuse
- `tickets/` - scenario markdown files
- `evidence/` - text-based evidence snippets
- `templates/` - student and instructor templates
- `instructor/` - answer key and marking guide
- `student-pack/` - learner-facing pack without direct answers
