# Student Quick Start

1. Open the assigned ticket in the `tickets/` folder or GitHub Issue.
2. Read the user problem carefully.
3. Review the attached evidence files.
4. Complete `templates/student-response-template.md`.
5. Submit your answer in the method set by your instructor:
   - issue comment
   - pull request
   - completed markdown file in your branch

## Remember the CompTIA A+ approach
- Identify the problem
- Establish a theory of probable cause
- Test the theory
- Establish a plan of action
- Implement the solution
- Verify full functionality
- Document findings and actions
